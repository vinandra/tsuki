# Tsuki

simple node.js & typescript project to save phone contacts

## How To Use

To clone and run this application,

```bash
# Clone this repository
$ git clone https://github.com/vinandra/tsuki

# Go into the directory file
$ yarn

# how to run
$ yarn dev
```
